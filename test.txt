Fix 2: Disable Keyring Permanently
The simplest way to disable the keyring prompt is to create an empty keyring with no password:
bash# Remove existing keyrings
rm -rf ~/.local/share/keyrings/*

# Create a new default keyring with empty password
mkdir -p ~/.local/share/keyrings

cat > ~/.local/share/keyrings/default << 'EOF'
[keyring]
display-name=Default keyring
ctime=0
mtime=0
lock-on-idle=false
lock-after=false
EOF

cat > ~/.local/share/keyrings/default.keyring << 'EOF'
[keyring]
display-name=Default keyring
ctime=0
mtime=0
lock-on-idle=false
lock-after=false
EOF

Fix 3: Alternative - Disable Keyring Daemon at Login
bash# Create autostart directory if it doesn't exist
mkdir -p ~/.config/autostart

# Disable gnome-keyring-daemon
cat > ~/.config/autostart/gnome-keyring-daemon.desktop << 'EOF'
[Desktop Entry]
Type=Application
Name=Disable Keyring
Exec=/bin/true
Hidden=false
NoDisplay=true
X-GNOME-Autostart-enabled=false
EOF
