3. Play Fullscreen Video in Web UI on Boot
Yes, you can autostart a fullscreen Chromium browser and load a webpage (local or online) with a video.

A. Create an HTML file:

Paste:

html
Copy
Edit
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Chatty AI Boot Video</title>
  <style>
    body, html {
      margin: 0; padding: 0;
      overflow: hidden;
      background: black;
    }
    video {
      width: 100vw;
      height: 100vh;
      object-fit: cover;
    }
  </style>
</head>
<body>
  <video autoplay muted loop>
    <source src="boot_intro.mp4" type="video/mp4">
  </video>
</body>
</html>
Place boot_intro.mp4 in that folder.

B. Install Chromium if needed:
bash
Copy
Edit

C. Autostart in full screen:
Create an autostart config:

bash
Copy
Edit
mkdir -p ~/.config/lxsession/LXDE-pi
nano ~/.config/lxsession/LXDE-pi/autostart
Add:

bash
Copy
Edit
@chromium-browser --kiosk --autoplay-policy=no-user-gesture-required file:///home/nickspi5/chatty_ui/index.html
On reboot, this will play the video fullscreen.
